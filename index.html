待确认 Checklist
A. 机票
  • 曼谷→昆明 5.29：起降机场 / 航司 / 含托运 / 可退改
  • 北京→昆明 5.29：同伴航班时间，尽量不晚 Xinyi 太多
  • 昆明深夜落地：酒店是否 24h 接机、是否提前联系司机
  • 丽江→北京 6.3：是否直飞、时间优先下午晚上
  • 到达机场是 PEK 还是 PKX
  • 北京→曼谷下周末回程：机票 / 托运 / 泰国入境证件
B. 高铁
  • 昆明→丽江：出发站是昆明站还是昆明南站
  • 机场酒店到高铁站车程，预留 1h+ 机动
  • 买 10:30 以后车次
  • 购票证件与护照/身份证一致；能否刷证进站 or 需取票
C. 丽江 ↔ 泸沽湖交通
  • 5.31 丽江→泸沽湖的车（且慢入影松或酒店确认）
  • 包车还是拼车；车型 / 人数 / 价格 / 出发时间
  • 是否酒店门口接、是否送到且慢入影松门口
  • 6.2 女神山后能否直接接上回丽江
  • 费用是否含高速费 / 停车费 / 司机餐补
D. 泸沽湖游玩
  • 6.1 环湖包车：价格 / 时长 / 停靠点
  • 是否含猪槽船 / 里格观景台 / 情人滩 / 草海 / 走婚桥
  • 6.2 女神山：索道开放时间 / 票价 / 是否提前买票
  • 女神山当天天气，雨/雾及时改轻松方案
  • 酒店能否安排早餐 / 行李寄存 / 出发接送
E. 住宿
  • 5.29 昆明机场酒店：接机 / 24h前台 / 取消政策
  • 5.30 丽江：别选古城最里面，车能否到门口附近
  • 5.31–6.2 且慢入影松：湖景房型 / 含早 / 帮忙安排交通
  • 6.2 丽江：安静舒服精品酒店（恢复夜）
  • 所有酒店确认入住人姓名 / 证件 / 入住时间 / 取消政策
F. 行李与健康
  • 防晒 / 墨镜 / 润唇膏 / 保湿
  • 薄外套/冲锋衣（泸沽湖丽江早晚温差大）
  • 晕车药（丽江→泸沽湖山路弯多）
  • 轻便运动鞋（女神山 + 古城石板路）
  • 高海拔的晚上别喝太多酒、别熬夜

整体判断

去程用昆明做缓冲，避免深夜到达后继续赶；5.31–6.2 泸沽湖有完整两晚，能看湖、环湖、上女神山；6.2 提前回丽江，把最累的山路提前消化掉；6.3 从丽江飞北京收尾不赶。
